Inventory API

Base API URL: https://officebali.e1-vhp.com:34443

API Endpoints

All endpoints use POST method with JSON request body.

1. Get Main Groups

POST /api/maingroup

Description: Retrieve all available main groups.

Request Body:


        {}

Response:


        {
          "data": [
            {
              "MainGroupNo": 1,
              "MainGroup": "FOOD"
            },
            {
              "MainGroupNo": 2,
              "MainGroup": "BEVERAGE"
            }
          ]
        }

cURL Example:


        curl -X POST https://officebali.e1-vhp.com:34443/api/maingroup \
        -H "Content-Type: application/json" \
        -d "{}"

2. Get Sub Groups

POST /api/subgroup

Description: Retrieve subgroups with optional filtering by main group.

Request Body:


        {
          "main_group_no": 0
        }

Request Properties:

Property Type Default Description
main_group_no int 0 Filter by MainGroupNo. Use 0 for all subgroups, or a specific number to filter (e.g., 1 for FOOD)

Response:


        {
          "data": [
            {
              "MainGroupNo": 1,
              "SubGroupNo": 1,
              "SubGroup": "Dairy Product"
            }
          ]
        }

cURL Examples:

Get all sub groups:


        curl -X POST https://officebali.e1-vhp.com:34443/api/subgroup \
        -H "Content-Type: application/json" \
        -d '{"main_group_no": 0}'

Get sub groups for main group 1:


        curl -X POST https://officebali.e1-vhp.com:34443/api/subgroup \
        -H "Content-Type: application/json" \
        -d '{"main_group_no": 1}'

3. Get Inventory

POST /api/inventory

Description: Retrieve inventory items with pagination and various filter options.

Request Body:


        {
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }

Request Properties:

Property Type Default Description
limit int 50 Number of items displayed per page
last_article_no int 0 Cursor for pagination. Use 0 for the first page, or the last ArticleNo from the previous response for the next page
main_group_no int 0 Filter by MainGroupNo. Use 0 for all, or a specific number (e.g., 1 for FOOD)
sub_group_no int 0 Filter by SubGroupNo. Use 0 for all, or a specific number
purchase_date_from string "" Filter items purchased from this date onwards. Use "" if no filter needed. Format: "(DD/MM/YYYY) 1/1/2023"
purchase_date_to string "" Filter items purchased up to this date. Use "" if no filter needed. Format: "(DD/MM/YYYY) 12/31/2023"
description string "" Search by description (partial match, case-insensitive). Use "" if no search needed

Response:


        {
          "data": [
            {
              "MainGroupNo": 1,
              "MainGroup": "FOOD",
              "SubGroupNo": 1,
              "SubGroup": "Dairy Product",
              "ArticleNo": 1010001,
              "Description": "Butter Portion Unsalted Elle & Vire",
              "Unit": "Pcs",
              "Price": 2997,
              "LastPurchaseDate": "4/15/2023"
            }
          ],
          "last_article_no": 1010001,
          "last_article": false
        }

Response Fields:

  • data: Array containing inventory items
  • last_article_no: ArticleNo of the last item in this response (use for next page)
  • last_article: Boolean indicating whether there are more items available

cURL Examples:

Get first page (default: all items, 50 per page):


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }'

Get next page (change last_article_no):


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 1010050,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }'

Filter by main group 1 (change main_group_no):
