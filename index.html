Inventory API

Base API URL: https://officebali.e1-vhp.com:34443

API Endpoints

Semua endpoint menggunakan metode POST dengan request body format JSON.

1. Get Main Groups

POST /api/maingroup

Description: Mengambil semua main group yang tersedia.

Request Body:


        {}

Response:


        {
          "data": [
            {
              "MainGroupNo": 1,
              "MainGroup": "FOOD"
            },
            {
              "MainGroupNo": 2,
              "MainGroup": "BEVERAGE"
            }
          ]
        }

cURL Example:


        curl -X POST https://officebali.e1-vhp.com:34443/api/maingroup \
        -H "Content-Type: application/json" \
        -d "{}"

2. Get Sub Groups

POST /api/subgroup

Description: Mengambil sub group dengan opsi filter berdasarkan main group.

Request Body:


        {
          "main_group_no": 0
        }

Request Properties:

Property Type Default Description
main_group_no int 0 Filter berdasarkan MainGroupNo. Gunakan 0 untuk semua sub group, atau nomor spesifik untuk filter tertentu (contoh: 1 untuk FOOD)

Response:


        {
          "data": [
            {
              "MainGroupNo": 1,
              "SubGroupNo": 1,
              "SubGroup": "Dairy Product"
            }
          ]
        }

cURL Examples:

Mengambil semua sub group:


        curl -X POST https://officebali.e1-vhp.com:34443/api/subgroup \
        -H "Content-Type: application/json" \
        -d '{"main_group_no": 0}'

Mengambil sub group untuk main group 1:


        curl -X POST https://officebali.e1-vhp.com:34443/api/subgroup \
        -H "Content-Type: application/json" \
        -d '{"main_group_no": 1}'

3. Get Inventory

POST /api/inventory

Description: Mengambil item inventory dengan pagination dan berbagai opsi filter.

Request Body:


        {
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }

Request Properties:

Property Type Default Description
limit int 50 Jumlah item yang ditampilkan per halaman
last_article_no int 0 Cursor untuk pagination. Gunakan 0 untuk halaman pertama, atau ArticleNo terakhir dari response sebelumnya untuk halaman berikutnya
main_group_no int 0 Filter berdasarkan MainGroupNo. Gunakan 0 untuk semua, atau nomor spesifik (contoh: 1 untuk FOOD)
sub_group_no int 0 Filter berdasarkan SubGroupNo. Gunakan 0 untuk semua, atau nomor spesifik
purchase_date_from string "" Filter item yang dibeli mulai dari tanggal ini. Gunakan "" jika tidak perlu filter. Format: "(DD/MM/YYYY) 1/1/2023"
purchase_date_to string "" Filter item yang dibeli sampai tanggal ini. Gunakan "" jika tidak perlu filter. Format: "(DD/MM/YYYY) 12/31/2023"
description string "" Pencarian berdasarkan deskripsi (partial match, case-insensitive). Gunakan "" jika tidak perlu pencarian

Response:


        {
          "data": [
            {
              "MainGroupNo": 1,
              "MainGroup": "FOOD",
              "SubGroupNo": 1,
              "SubGroup": "Dairy Product",
              "ArticleNo": 1010001,
              "Description": "Butter Portion Unsalted Elle & Vire",
              "Unit": "Pcs",
              "Price": 2997,
              "LastPurchaseDate": "4/15/2023"
            }
          ],
          "last_article_no": 1010001,
          "last_article": false
        }

Response Fields:

  • data: Array berisi item-item inventory
  • last_article_no: ArticleNo dari item terakhir di response ini (gunakan untuk halaman berikutnya)
  • last_article: Boolean yang menunjukkan apakah masih ada item yang tersedia

cURL Examples:

Mengambil halaman pertama (default: semua item, 50 per halaman):


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }'

Mengambil halaman berikutnya (ubah last_article_no):


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 1010050,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }'

Filter berdasarkan main group 1 (ubah main_group_no):


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 1,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": ""
        }'

Pencarian berdasarkan deskripsi "Water":


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "",
          "purchase_date_to": "",
          "description": "Water"
        }'

Filter berdasarkan rentang tanggal pembelian:


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 50,
          "last_article_no": 0,
          "main_group_no": 0,
          "sub_group_no": 0,
          "purchase_date_from": "1/1/2023",
          "purchase_date_to": "12/31/2023",
          "description": ""
        }'

Filter kombinasi (main group 1, sub group 1, dengan pencarian deskripsi):


        curl -X POST https://officebali.e1-vhp.com:34443/api/inventory \
        -H "Content-Type: application/json" \
        -d '{
          "limit": 20,
          "last_article_no": 0,
          "main_group_no": 1,
          "sub_group_no": 1,
          "purchase_date_from": "1/1/2023",
          "purchase_date_to": "",
          "description": "Butter"
